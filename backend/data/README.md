# Data Folder

Place datasets here before training.

Expected disease diagnosis CSV:

```csv
symptoms,disease
"fever cough body pain","Flu"
"increased thirst frequent urination fatigue","Diabetes"
```

Recommended sources:

- Kaggle Disease Diagnosis/Symptom Dataset for symptom-to-disease labels.
- Samanantar dataset for multilingual sentence pairs.

For production, validate labels with a medical professional and split data into train, validation, and test sets.
